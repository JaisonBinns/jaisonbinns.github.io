Headset Shop Site

Summary: Created a fully functional shop site that can be used a template and a reference for future freelance work.


Project Details

This shop site includes the following features email authentication, price and size filters, search feature, upload product with multiple images, continously load more products, paypal payment integration, and payment history. This project has given me more confidence to create full stack applications. With this project, I have taken a deeper dive into state management (compared to my SimpleStream project and test repos), gaining confidence and working proficiency. This project took a bit longer than I expected. I understand why some do not like using redux unless it is necessary. The tedious code wiring from components to actions/reducers made me realize why redux should not necessarily be an immediate "must-have" for an application that needs state management.

Vibespot

Summary: Redesigned the UX/UI of the app, resulting in 2,000 app downloads and 10 corporate parterships.


Project Details

I was a founding member for this company. Vibespot is a social platform to discover new places and meet new people. I focused on business development and UI/UX design. While developing partnerships, sponsorships, and promotional deals, I kept a log of the critical user feedback I received from the early users to redesign the UI/UX to the state shown in the video. For this project, I used Microsoft Paint for the wireframe and InVision to create the interactive prototype. The invision prototype helped us continue pitching investors and secure funding while version 2.0 was being developed. Ultimately, my experience with Vibespot is what led to my transition into software developer. Because I was a part of pre-seed company, I assisted in building the roadmap. I imagined, pitched, and designed the "Venue Card" and the "Filter Cities & Date" features. I discovered I want to develop the software.

Vibespot is available for download. Source code is not available for this project.
